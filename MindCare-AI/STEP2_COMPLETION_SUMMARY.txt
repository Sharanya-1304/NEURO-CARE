📊 PIPELINE OVERVIEW
═════════════════════════════════════════════════════════════════════════════════

Raw Data (CSV)
    ↓
LOAD (load_data.py)
├─ Read CSV file
├─ Check structure
└─ Display basic stats
    ↓
Loaded Data
    ↓
EXPLORE (load_data.py)
├─ Check data types
├─ Find patterns
├─ Check missing values
└─ Check duplicates
    ↓
CLEAN (clean_data.py)
├─ Remove duplicates
├─ Fill missing values
├─ Encode categories (Male→0, Female→1)
└─ Remove outliers
    ↓
Clean Data
    ↓
FEATURE ENGINEERING
├─ Select relevant features
├─ Split into X (features) & y (target)
└─ 80% training, 20% testing
    ↓
TRAIN (train_model.py)
├─ Initialize Random Forest
├─ Train on 80% of data
└─ Create 100 decision trees
    ↓
EVALUATE
├─ Test on remaining 20%
├─ Calculate accuracy (~85%)
├─ Calculate precision, recall, F1
└─ Feature importance analysis
    ↓
SAVE
├─ Save model: models/stress_model.pkl
├─ Save encoders: models/encoders/
└─ Save feature names: models/feature_names.pkl
    ↓
PREDICT (predict.py)
├─ Load saved model
└─ Make predictions on new data
