📈 UNDERSTANDING THE METRICS
═════════════════════════════════════════════════════════════════════════════════

Accuracy (Most Important)
   Definition: % of correct predictions overall
   Formula: Correct Predictions / Total Predictions
   Example: 85% = 850 out of 1000 correct
   Target: 85%+
   Interpretation: "My model is right 85% of the time"
   Normal Range: 80-88% (varies based on data)

Precision
   Definition: Of positive predictions, how many were correct?
   Formula: TP / (TP + FP)
   Interpretation: "When I say HIGH STRESS, am I right?"
   Best When: False alarms are costly

Recall
   Definition: Of actual positives, how many did we find?
   Formula: TP / (TP + FN)
   Interpretation: "Did I catch all the HIGH STRESS cases?"
   Best When: Missing cases is costly

F1-Score
   Definition: Balance between Precision and Recall
   Formula: 2 × (Precision × Recall) / (Precision + Recall)
   Range: 0 to 1 (higher is better)
   Interpretation: "Overall, how balanced is my model?"


🔧 TROUBLESHOOTING - QUICK FIXES
═════════════════════════════════════════════════════════════════════════════════

Problem: "FileNotFoundError: data/raw/student_depression_dataset.csv"
Solution:
   1. Download from Kaggle: "Student Depression Dataset"
   2. Check file exists: ls -la data/raw/
   3. Verify file name matches exactly

Problem: "ModuleNotFoundError: No module named 'xgboost'"
Solution:
   Run: pip install -r requirements.txt
   Or: pip install xgboost lightgbm plotly scikit-learn pandas numpy joblib

Problem: "Jupyter notebook command not found"
Solution:
   Run: pip install jupyter
   Then: jupyter notebook

Problem: "Accuracy below 85%"
Solution:
   This is NORMAL! Typical range is 80-88%
   Depends on dataset quality
   Not a failure - proceed to next step
   You can optimize later

Problem: "Model file is empty / too small"
Solution:
   Check file size: ls -lh models/stress_model.pkl
   Should be: 1-2 MB
   If smaller: Training failed, check console output
